\doxysubsection{Detailed Description}
Morphs attach themselves to Gear objects, so any Gear can have an morph applied to it, although it may not have any visible change. Primary candidates for morphs are. 


\begin{DoxyCode}{0}
\DoxyCodeLine{geng::Banners}

\end{DoxyCode}
 and 
\begin{DoxyCode}{0}
\DoxyCodeLine{geng::Sprites}

\end{DoxyCode}
.

To create a new morph, make a subclass of the morph object and add it via the Engine. Morphs have six member variables\+:
\begin{DoxyEnumerate}
\item Transform2\+D\texorpdfstring{$\ast$}{*} t -\/\texorpdfstring{$>$}{>} pointer to the object\textquotesingle{}s Transform2D \mbox{[}protected\mbox{]}
\item float init\+Duration -\/\texorpdfstring{$>$}{>} initial duration of the morph (-\/1 implies permanent) \mbox{[}protected\mbox{]}
\item float duration -\/\texorpdfstring{$>$}{>} current duration of the morph \mbox{[}protected\mbox{]}
\item bool permanent -\/\texorpdfstring{$>$}{>} if duration is set to -\/1, permanent is set to true in the morph class. \mbox{[}protected\mbox{]}
\item int id -\/\texorpdfstring{$>$}{>} for engine identification and speed purposes \mbox{[}private\mbox{]}
\item int target\+\_\+id -\/\texorpdfstring{$>$}{>} lets us remove morph by object ID, since Objects don\textquotesingle{}t have an morph pointer \mbox{[}private\mbox{]}
\end{DoxyEnumerate}

\begin{DoxyNote}{Note}
Notes for making morphs\+:
\begin{DoxyItemize}
\item \doxylink{classgeng_1_1_morph}{Morph} is a pure virtual class
\item \doxylink{classgeng_1_1_morph}{Morph}\textquotesingle{}s constructors will initialize the init\+Duration and permanent fields for you
\item Add morphs to the \doxylink{classgeng_1_1_morph_manager}{Morph\+Manager} with 
\begin{DoxyCode}{0}
\DoxyCodeLine{engine.add\_morph(\textcolor{keyword}{new}\ geng::Morph(...))\ }

\end{DoxyCode}

\item Remove permanent morphs with 
\begin{DoxyCode}{0}
\DoxyCodeLine{engine.remove\_morph(\mbox{\hyperlink{classgeng_1_1_morph_a716e15fe7d1545c4994135c8527833c9}{Morph}}\ *e)\ }

\end{DoxyCode}
 or 
\begin{DoxyCode}{0}
\DoxyCodeLine{engine.remove\_morph(Gear*\ g)\ }

\end{DoxyCode}

\item Unless you explicitly program it, Morphs will not return an object to its original transform upon completion
\end{DoxyItemize}
\end{DoxyNote}
\begin{DoxyWarning}{Warning}
To make an morph you must override the \doxylink{classgeng_1_1_morph_a09227482da74b711ffe0e4dc2d74c1da}{update()} function. Update() returns true if the morph is done, false otherwise. 
\end{DoxyWarning}
