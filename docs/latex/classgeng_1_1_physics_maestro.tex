\doxysubsection{Detailed Description}
The \doxylink{classgeng_1_1_physics_maestro}{Physics\+Maestro} basically handles everything physics in the engine, from updating colliders to determining collisions. A lot of the \doxylink{classgeng_1_1_physics_maestro}{Physics\+Maestro} code is going to be hot-\/swappable so you don\textquotesingle{}t need to use the engine defaults, and can customize according to your whims.
